Welcome to Cradlepoint Inventory Management

This system is designed to streamline the management of Cradlepoint devices, providing administrators with tools to monitor, search, and update their inventory efficiently. Below is a brief overview of the system's features:

  • Admin Dashboard: Manage admin users, reset the database, and maintain control over access.
  • Full Inventory: View all Cradlepoint-related tables in one place for a comprehensive overview.
  • Router Status Lookup: Fetch the current status of a router via the NetCloud Manager (NCM) API.
  • Router Detail Finder: Search for detailed Cradlepoint router information using a MAC address.
  • Add a Router: Add new routers to the database, ensuring unique MAC addresses. (Admin-only access)
  • Admin Login: Secure access for administrators to manage system features and data.

Use the navigation bar above to access each section. For help or additional instructions, refer to the README.
